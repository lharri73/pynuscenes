An Iterable Dataloader for nuScenes
===================================
.. toctree::
   :maxdepth: 2
   :hidden:

   src/config
   src/usage
   src/autodoc

**PyNuscenes** is a dataloader for the `NuScenes dataset <https://www.nuscenes.org/nuscenes>`_.
It uses the `NuScenes devkit <https://github.com/nutonomy/nuscenes-devkit>`_ and provides APIs for loading sensor data in different coordinate systems.
